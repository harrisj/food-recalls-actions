020-2011 Low - Class II Unreported Allergens Mislabeling

Minnesota Firm Recalls Honey Cured Turkey Stick Products Due To Mislabeling and an Undeclared Allergen

Monogram Meat Snacks, LLC
Closed
Fri, 03/11/2011 - Sat, 06/02/2012
California, Colorado, Illinois, Missouri, Ohio

WASHINGTON, March 11, 2011 - Monogram Meat Snacks, LLC, a Chandler, Minn., establishment is recalling approximately 5,125 pounds of Hannah's Honey Cured Turkey Sticks because they contain an undeclared allergen, wheat, which is not declared on the label. Wheat is a known allergen. The products...

Impacted Products
100 count cases containing 1-oz. packages of "HANNAH'S HONEY CURED TURKEY STICK." These products have an identifying case code of "706179" and the following package codes (use by/sell by dates): 09/09/11 and 10/01/11. 560 count cases containing 1-oz. packages of "HANNAH'S HONEY CURED TURKEY STICK." These products have an identifying case code of "330074" and the package code (use by/sell by date) of 10/01/11.
Quantity Recovered
3,839 pounds
