Active

• 32-oz. box containing “A TU GUSTO PALADAR POLLO (CHICKEN) CROQUETAS (CROQUETTES).” • 32-oz. box containing “A TU GUSTO PALADAR CHORIZO (SAUSAGE) CROQUETAS (CROQUETTES).” • 32-oz. box containing “A TU GUSTO PALADAR PESCADO (FISH) CROQUETAS (CROQUETTES).” • 32-oz. box containing “A TU GUSTO PALADAR JAMON (HAM) CROQUETAS (CROQUETTES).”
